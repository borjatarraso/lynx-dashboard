# Keybindings

The dashboard reuses the same keybinding vocabulary as the rest of the
suite so muscle memory carries over.

## Global shortcuts

| Key            | Where     | Action                                     |
|----------------|-----------|--------------------------------------------|
| `q`            | TUI, REPL | Quit                                       |
| `Ctrl+Q`       | all       | Quit                                       |
| `Esc`          | all       | Close the topmost modal                    |
| `a`            | TUI       | Open About dialog                          |
| `?` / `h`      | TUI, REPL | Open keybindings cheat-sheet               |
| `F1`           | GUI       | Open keybindings cheat-sheet               |
| `PgUp`/`PgDn`  | all       | Scroll long output (shared with core)      |
| `Ctrl+Home`/`Ctrl+End` | TUI | Jump to top/bottom of the focused scroll |

## Dashboard-specific

| Key   | Action                                                      |
|-------|-------------------------------------------------------------|
| `r`   | Recommend an agent for a company (opens dialog)             |
| `m`   | Cycle launch mode: tui → interactive → gui → console        |
| `t`   | Toggle run mode: production ↔ testing                       |
| `e`   | Easter egg                                                  |
| `/`   | (TUI) Quick search / filter within the focused table         |
